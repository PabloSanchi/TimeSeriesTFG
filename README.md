# Requirements
The user must have in the folder the following files:
- `requirements.txt`
- `serviceAccountKey.json`
- If using docker:
  - Software: `Docker`
- If deploying locally
  - Programming environment: `Python`
# RUN DOCKER COMPOSE
Command: `docker compose up -d`
# RUN LOCALLY
## Steps
- `pip3 install -r requirements.txt`
- `python3 main.py`
